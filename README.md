# Job4j DreamJob

Учебный проект на Spring Boot для работы с вакансиями. 
Реализован контроллер вакансий, отображение списка через Thymeleaf шаблоны, 
использование фрагментов (header/footer), а также работа с репозиторием в памяти. 
Проект предназначен для практики MVC архитектуры, 
шаблонизатора Thymeleaf и базовой структуры веб-приложений на Java.
